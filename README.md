## Build Setup:

```bash
# Download repository:
git clone https://github.com/vedees/react-cv-builder-tocode react-cv-builder

# Go to the app:
cd react-cv-builder

# Install dependencies:
npm install

# Server with hot reload at http://localhost:8080/
npm run dev

# Output will be at dist/ folder
npm run build
```
